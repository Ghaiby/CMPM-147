Experiment 03 - Alternate Worlds

Description

I created two auto generated tile maps, one for a dungeon, and one for an overworld

Technical

Implemented custom logic to generate the grid of characters to match the structures of a dungeon or overworld. Dungeon using rooms that connect and overworld using lakes made with noise

Reflection

This experiment took many failures and ambitious attempts and I am happy that I stuck it out for the final result. All the logic used to connect and generate rooms and render the walls is logic that I made through trial and error.
